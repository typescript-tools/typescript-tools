> Pin lerna dependencies to latest managed version

## Install

``` shell
npm install --save-dev @typescript-tools/pin-lerna-package-versions
```

## Use

``` shell
Usage:
    pin-lerna-package-versions <root>

Options:
    root    Root of lerna mono-repository
```
