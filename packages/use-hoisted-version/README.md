> Update lerna package to use hoisted version of npm dependency

## Install

```shell
npm install --save-dev @typescript-tools/use-hoisted-version
```

## Use

```shell
Usage:
    use-hoisted-version <package>

Options:
    package    Package for which to update dependencies
```
